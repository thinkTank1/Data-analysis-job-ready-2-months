## 4. SQL for Data Analysis

### Level 7: Introduction to SQL
**Video Tutorial:** [SQL Tutorial for Beginners](https://www.youtube.com/watch?v=HXV3zeQKqGY)  
**Step-by-Step Guide:**
1. Download and install MySQL Community Server from [here](https://dev.mysql.com/downloads/mysql/).
   - Step 1: Go to the MySQL download page.
   - Step 2: Choose your operating system and download the installer.
   - Step 3: Run the installer and follow the prompts to install MySQL.

2. Install MySQL Workbench and connect to your local MySQL server.
   - Step 1: Download MySQL Workbench from the same page.
   - Step 2: Install MySQL Workbench.
   - Step 3: Open MySQL Workbench and click on the "+" icon next to MySQL Connections.
   - Step 4: Give your connection a name and click "Test Connection" to make sure it works.

3. Create a new database called "store" and a table called "products" with columns for id, name, price, and category. Insert at least 10 products into this table.
   - Step 1: In MySQL Workbench, write this SQL command: `CREATE DATABASE store;`
   - Step 2: Click the lightning bolt icon to run the command.
   - Step 3: Write and run this command to create the products table:
     ```sql
     CREATE TABLE products (
         id INT PRIMARY KEY,
         name VARCHAR(100),
         price DECIMAL(10, 2),
         category VARCHAR(50)
     );
     ```
   - Step 4: Insert products using commands like this:
     ```sql
     INSERT INTO products (id, name, price, category)
     VALUES (1, 'T-shirt', 19.99, 'Clothing');
     ```
   - Step 5: Repeat step 4 to insert at least 10 products.

**Tasks:**
1. Create a new table called "sales" with columns for id, product_id, sale_date, and quantity. Insert at least 20 sales records.
   - Step 1: Write and run this SQL command to create the sales table:
     ```sql
     CREATE TABLE sales (
         id INT PRIMARY KEY,
         product_id INT,
         sale_date DATE,
         quantity INT
     );
     ```
   - Step 2: Insert sales records using commands like this:
     ```sql
     INSERT INTO sales (id, product_id, sale_date, quantity)
     VALUES (1, 1, '2023-09-01', 5);
     ```
   - Step 3: Repeat step 2 to insert at least 20 sales records.

2. Write a SQL query to find the total sales amount for each product category.
   - Step 1: Write this SQL query:
     ```sql
     SELECT p.category, SUM(p.price * s.quantity) as total_sales
     FROM products p
     JOIN sales s ON p.id = s.product_id
     GROUP BY p.category;
     ```
   - Step 2: Run the query and examine the results.

3. Write a SQL query to find the top 5 selling products.
   - Step 1: Write this SQL query:
     ```sql
     SELECT p.name, SUM(s.quantity) as total_quantity
     FROM products p
     JOIN sales s ON p.id = s.product_id
     GROUP BY p.id, p.name
     ORDER BY total_quantity DESC
     LIMIT 5;
     ```
   - Step 2: Run the query and examine the results.

### Level 8: Advanced SQL for Data Analysis
**Video Tutorial:** [Advanced SQL Tutorial](https://www.youtube.com/watch?v=2Fn0WAyZV0E)  
**Tasks:**
1. Practice joins, aggregate functions, and subqueries using the "store" database.
   - Step 1: Write a query to find the average price of products in each category:
     ```sql
     SELECT category, AVG(price) as avg_price
     FROM products
     GROUP BY category;
     ```
   - Step 2: Write a query to find the total sales for each product:
     ```sql
     SELECT p.name, SUM(p.price * s.quantity) as total_sales
     FROM products p
     JOIN sales s ON p.id = s.product_id
     GROUP BY p.id, p.name;
     ```
   - Step 3: Write a query to find products that have never been sold:
     ```sql
     SELECT name
     FROM products
     WHERE id NOT IN (SELECT DISTINCT product_id FROM sales);
     ```
   - Step 4: Run each query and examine the results.

     ```python
     import matplotlib.pyplot as plt
     ```

     ```python
     df['date'] = pd.to_datetime(df['date'])
     ```

     ```python
     daily_sales = df.groupby('date')['sales'].sum()
     ```

     ```python
     plt.figure(figsize=(12, 6))

     plt.xlabel('Date')
     plt.ylabel('Total Sales')
     plt.xticks(rotation=45)
     plt.tight_layout()
     ```
