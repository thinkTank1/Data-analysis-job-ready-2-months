## 4. SQL for Data Analysis

### Level 7: Introduction to SQL
**Video Tutorial:** [SQL Tutorial for Beginners](https://www.youtube.com/watch?v=HXV3zeQKqGY)  
**Step-by-Step Guide:**
1. Download and install MySQL Community Server from [here](https://dev.mysql.com/downloads/mysql/).
2. Install MySQL Workbench and connect to your local MySQL server.
3. Create a new database called "store" and a table called "products" with columns for id, name, price, and category. Insert at least 10 products into this table.

**Tasks:**
1. Create a new table called "sales" with columns for id, product_id, sale_date, and quantity. Insert at least 20 sales records.
2. Write a SQL query to find the total sales amount for each product category.
3. Write a SQL query to find the top 5 selling products.

### Level 8: Advanced SQL for Data Analysis
**Video Tutorial:** [Advanced SQL Tutorial](https://www.youtube.com/watch?v=2Fn0WAyZV0E)  
**Tasks:**
1. Practice joins, aggregate functions, and subqueries using the "store" database.
